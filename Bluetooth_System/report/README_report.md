## How to Generate the PDF Report

1. Open each `.md` file in the `docs/` folder.
2. Copy content into a Word document or Google Docs.
3. Apply college report formatting (12pt Times New Roman, 1.5 line spacing, 1-inch margins).
4. Add your institution's cover page, certificate, and acknowledgement pages.
5. Export as PDF.
6. Save as `mini_project_report.pdf` in this folder.
