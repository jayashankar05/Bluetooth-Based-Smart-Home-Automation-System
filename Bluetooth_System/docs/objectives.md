# Objectives

## 2. Objectives of the Project

### 2.1 Primary Objectives

1. **To design** a low-cost wireless home automation system using Bluetooth technology and the Arduino UNO microcontroller.
2. **To interface** the HC-05 Bluetooth module with Arduino UNO using the SoftwareSerial communication protocol.
3. **To implement** relay-based switching to control up to four AC electrical appliances independently.
4. **To develop** a character-command protocol for sending control instructions from a smartphone to the Arduino.
5. **To verify** the working range and response reliability of the Bluetooth communication link.

### 2.2 Secondary Objectives

1. **To demonstrate** the concepts of UART serial communication, digital I/O control, and relay interfacing in an integrated embedded system.
2. **To produce** complete technical documentation including circuit diagrams, code explanation, and a project report suitable for academic submission.
3. **To lay the foundation** for future extensions such as Wi-Fi-based control, voice commands, or sensor-driven automation.

### 2.3 Learning Outcomes

Upon completion of this project, the student will be able to:

| # | Learning Outcome |
|---|-----------------|
| 1 | Understand the operation of the HC-05 Bluetooth module in slave mode |
| 2 | Interface SoftwareSerial in Arduino for secondary UART communication |
| 3 | Interface a 4-channel relay module with a microcontroller |
| 4 | Understand active-LOW relay logic and safe AC appliance switching |
| 5 | Write structured, well-commented embedded C code for Arduino |
| 6 | Test and debug a hardware prototype systematically |
| 7 | Document a hardware project in a professional GitHub repository format |
